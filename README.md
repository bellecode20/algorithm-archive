# algorithm-archive

팀 알고리즘 스터디 기록 저장소입니다.  
평일 1문제 풀이를 목표로 하며, **월별 관리 + 개인 폴더 구조**로 운영합니다.

---

## 📌 스터디 규칙

- 평일 하루 1문제 풀이
- 문제 풀이 후 당일 커밋 원칙

---

## 📁 폴더 구조

```plaintext
이름/
└── YYYY-MM/
    └── DD_플랫폼_문제명.확장자
```

🧾 커밋 컨벤션 (Commit Convention)
기본 포맷
[YYMMDD][이름] 플랫폼 문제명

예시
[251222][su] codetree 격자 숫자 놀이
[251223][yena] boj 배열 연산
